# 🌡️ Temperature Conversion Program

## 📖 Project Overview

The Temperature Conversion Program is a simple web application developed using HTML, CSS, and JavaScript. Its purpose is to help users convert temperature values between Celsius and Fahrenheit quickly and easily. The application provides a clean and user-friendly interface where users can enter a temperature value, choose the desired conversion type, and instantly view the result.

---

## 🎯 Project Objective

The main objective of this project is to practice the fundamental concepts of front-end web development, including:

✨ Creating forms with HTML
✨ Styling web pages with CSS
✨ Using JavaScript to perform calculations
✨ Manipulating HTML elements dynamically
✨ Handling user interactions and events

---

## 🏗️ Application Structure

The application consists of three main files:

📄 **HTML File**: Defines the structure and content of the webpage.

🎨 **CSS File**: Controls the appearance and layout of the application.

⚙️ **JavaScript File**: Implements the temperature conversion logic and updates the displayed result.

---

## 🖥️ User Interface

The interface is designed as a centered card containing:

🔹 A title displaying the name of the application.

🔹 An input field where the user enters a temperature value.

🔹 Two radio buttons allowing the user to choose the conversion type:

* Celsius to Fahrenheit
* Fahrenheit to Celsius

🔹 A submit button that starts the conversion process.

🔹 A result area where the converted temperature is displayed.

The layout is simple, clean, and easy to understand, making the application accessible for all users.

---

## 🎨 Design and Styling

The visual design focuses on simplicity and readability.

The page uses a light gray background to create a soft and modern appearance. The conversion form is displayed in the center of the page with rounded corners and a shadow effect, giving it a card-like design.

The title is displayed in blue to attract attention, while the submit button uses a red color that changes to a darker shade when the user hovers over it. This hover effect improves the user experience by providing visual feedback.

The result text is centered and displayed in a larger font size to make it clearly visible.

---

## ⚙️ Application Functionality

When the user enters a temperature value and selects a conversion option, the application performs the corresponding calculation.

If the user selects **Celsius to Fahrenheit**, the application converts the entered Celsius value into Fahrenheit.

If the user selects **Fahrenheit to Celsius**, the application converts the entered Fahrenheit value into Celsius.

After the calculation is completed, the result is displayed immediately on the page without requiring a page refresh.

If the user clicks the submit button without selecting a conversion type, the application displays a message asking the user to select a unit.

---

## 💡 JavaScript Logic

The JavaScript code retrieves the values entered by the user and checks which conversion option has been selected.

A temporary variable is used to store the temperature value and the conversion result during the calculation process.

The application then updates the content of the result area dynamically, allowing the user to see the converted temperature instantly.

The displayed result is rounded to one decimal place to improve readability and provide a cleaner presentation.
